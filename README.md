# OpenPanelFlutter

**OpenPanelFlutter** is a Python-based structural and aeroelastic solver designed for the analysis of thin-walled structures and composite skew panels.

## Overview

This project focuses on the implementation of nonlinear structural mechanics and aeroelasticity models. It is being developed to support advanced research in aerospace engineering, specifically targeting:

*   **Nonlinear Structural Analysis:** Moderate geometric nonlinearities.
*   **Composite Materials:** Modeling of laminated plates using specialized failure criteria (e.g., Hashin-Rotem).
*   **Aeroelasticity:** Coupling structural models with supersonic aerodynamic loads for stability and flutter analysis.

## Core Features (In Development)

*   **Nonlinear Solver:** Efficient routines for solving large-scale deformation problems in thin-walled structures.
*   **Material Library:** Support for orthotropic and composite material properties.
*   **Visual Analysis:** Integration with Matplotlib for displacement, stress, and flutter boundary plotting.

## Project Structure

*   `src/`: Main source code for the solver and aerodynamic models.
*   `examples/`: Sample scripts demonstrating panel analysis.

## Academic Context

This software is part of a Master's research at the **Instituto Tecnológico de Aeronáutica (ITA)** in Brazil.
